Типы лицензий
Источник: wiki → Лицензирование
НЕАКТИВНАЯ
Серая лицензия
Отработала срок действия. Устройство перестаёт воспроизводить контент. Требует продления или замены.
БЕССРОЧНАЯ
Синяя лицензия
Выкупается клиентом навсегда. Без даты истечения. Разовая инвестиция — идеально для долгосрочных инсталляций.
ПОДПИСКА
Зелёная лицензия
Фиксированный срок действия с датой окончания. Продлевается. Оптимально для рекламных сетей.
ПРОБНАЯ
Trial лицензия
Короткий срок действия для тестирования. Отличается от платной описанием и сроком. Для ознакомления клиента.
⚠ Важно: каждое устройство = отдельная лицензия
Манипуляции с лицензиями ведут к блокировке всего аккаунта компании. Цены — только по запросу у менеджера SmartPlayer.

Серверная инфраструктура
  • Облако Россия, Казахстан, Индия, США · cms.smartplayer.org
  • Локальный сервер Ubuntu Server 24.04 LTS · Astra Linux 1.7 / 1.8
  • Виртуализация VMware (VMDK-образ) · Hyper-V (доп. оплата)
  • Кластер Master + Worker-ноды · до 5000 устройств · S3-хранилище
  • Мощности 200 экранов → 4 ядра / 8 ГБ · 5000 экранов → 12 ядер / 52 ГБ
  • Порты 80 (HTTP) / 443 (HTTPS) · WebSocket WS/WSS · GlobalSign для HTTPS

  • OpenCV Бесплатно · Win10 + Ubuntu · наименее точный

⚙️
Аппаратные интеграции
  • Matrox Только Windows · специальная сборка SmartPlayer · запрашивать у менеджера
  • RFID Touch&Watch Только Windows · считыватель RFID-меток + триггеры
  • Последовательные порты COM · RS-232 · RS-485 · RS-422 (все платформы кроме BrightSign)
  • GPIO Триггеры по входу/выходу
  • Wake-on-LAN Удалённое включение выключенных устройств
  • HDMI CEC НЕ поддерживается — только RS-232 для управления ТВ
